# Resume Generator

A full-stack resume tailoring app built with React, Vite, TypeScript, FastAPI, and Google Gemini API.

## Features

- Upload resume files in PDF, DOCX, TXT, and RTF
- Paste a job description
- Generate a tailored resume
- Save the generated resume as DOCX
- Optional resume customization with checkboxes

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Material UI

### Backend
- FastAPI
- Python
- Google Gemini API
- pypdf
- python-docx
- striprtf
